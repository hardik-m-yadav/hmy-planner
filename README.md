# Hmy Planner

A modern AI-powered productivity and study planner built with React, Firebase, Redux Toolkit, and Tailwind CSS.

## Features

- Task Management
- Smart Notes
- Calendar View
- AI Planner
- Analytics Dashboard
- Achievement Badges
- User Authentication
- Dark / Light Mode
- Responsive Design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Firebase
- Redux Toolkit
- React Router
